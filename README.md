## :bangbang: Folder Structure

Here is the folder structure of this app.

```bash
apple-clone/
  |- public/
    |-- assets/
    |-- models/
    |-- apple-icon.png
    |-- favicon.ico
    |-- favicon1.png
    |-- favicon2.png
  |- src/
    |-- assets/
        |--- index.js
    |-- components/
        |--- Features.jsx
        |--- Footer.jsx
        |--- Hero.jsx
        |--- Highlights.jsx
        |--- HowItWorks.jsx
        |--- IPhone.jsx
        |--- Lights.jsx
        |--- Loader.jsx
        |--- Model.jsx
        |--- ModelView.jsx
        |--- Navbar.jsx
        |--- VideoCarousel.jsx
    |-- constants/
        |--- index.js
    |-- utils/
        |--- animations.js
    |-- App.css
    |-- App.jsx
    |-- index.css
    |-- main.jsx
  |- .eslintrc.json
  |- .gitignore
  |- index.html
  |- package-lock.json
  |- package.json
  |- postcss.config.js
  |- tailwind.config.js
  |- vite.config.js
```
